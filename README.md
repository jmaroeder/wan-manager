# wan-manager

Tool for monitoring ISP and performing tasks when it switches

## Getting started

1. [Install Poetry](https://python-poetry.org/docs/#installation)
2. Install dependencies:
   ```shell
   $ poetry install
   ```
3. Run wan-manager:
   ```shell
   $ poetry run wan-manager
   ```
